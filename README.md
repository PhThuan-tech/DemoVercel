# Pulse Studio Demo

Demo website tĩnh có tương tác, phù hợp để deploy trực tiếp lên Vercel.

## Chạy local

Mở `index.html` trong trình duyệt, hoặc chạy một static server bất kỳ.

## Deploy Vercel

1. Push thư mục này lên GitHub.
2. Import repository trong Vercel.
3. Giữ mặc định framework là `Other`.
4. Deploy.

Không cần build command hay output directory.
